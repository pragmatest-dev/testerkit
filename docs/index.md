## Quick Start

**Run the demo:**
```bash
cd demo && python run_demo.py
```

**Start the UI:**
```bash
litmus serve
```

**Configure for Claude Code:**
```bash
litmus setup claude-code
```

## Key Features

- **pytest integration** вҖ” Use familiar pytest patterns with hardware
- **Config-driven** вҖ” YAML configuration, Pydantic validation
- **Capability matching** вҖ” Automatically match products to compatible stations
- **Simulated mode** вҖ” Develop without hardware
- **AI-ready** вҖ” MCP server for Claude Code, Cursor, Cline
- **Event log** вҖ” Typed event stream with Arrow IPC storage and DuckDB queries
- **Channel store** вҖ” Time-series instrument data with LTTB decimation
- **Parquet storage** вҖ” Efficient columnar storage for analytics
- **Live monitoring** вҖ” Real-time event subscriptions via Arrow Flight

```
в”ңв”Җв”Җ config/          # Configuration models and loaders

в”ңв”Җв”Җ data/            # Event log, channels, Parquet backend
в”ңв”Җв”Җ mcp/             # MCP server
в”ңв”Җв”Җ api/             # HTTP API (FastAPI)

в””в”Җв”Җ client.py        # Python client library
```
